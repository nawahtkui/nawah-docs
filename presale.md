# البيع المبدئي لـ Nawah Token (NWTK)

## نظرة عامة
البيع المبدئي (Presale) هو الفرصة الأولى للمستثمرين لشراء عملة NWTK قبل طرحها في الأسواق العامة.

## تفاصيل البيع المبدئي

| العنصر | التفاصيل |
|--------|----------|
| **الرمز** | NWTK |
| **السلسلة** | Binance Smart Chain (BEP-20) |
| **الإجمالي** | 100,000,000 NWTK |
| **المعروض للبيع** | 20,000,000 NWTK (20%) |
| **سعر البيع** | $0.10 - $0.50 لكل NWTK |
| **الحد الأدنى للمساهمة** | $50 |
| **الحد الأقصى للمساهمة** | $10,000 |
| **عملات القبول** | BNB, USDT, (لاحقاً: عملات محلية) |

## مراحل البيع

### المرحلة الأولى: القائمة البيضاء (Whitelist)
- التسجيل المسبق للمستثمرين المهتمين
- مزايا: سعر مخفض، أولوية الشراء

### المرحلة الثانية: البيع الخاص (Private Sale)
- للمستثمرين الكبار والمؤسسات
- مدة: أسبوعين
- سعر: $0.10

### المرحلة الثالثة: البيع العام (Public Presale)
- مفتوح للجميع
- مدة: 4 أسابيع
- سعر: $0.25 - $0.50 (يزيد كل أسبوع)

## آلية العمل
1. يسجل المستثمر في القائمة البيضاء
2. بعد الموافقة، يقوم بتحويل العملات إلى عقد البيع الذكي
3. يستلم NWTK فور انتهاء البيع
4. يمكن المطالبة (Claim) بالرموز بعد الإدراج في البورصات

## ضمانات وأمان
- العقد الذكي مدقق من شركة أمان مستقلة
- قفل سيولة لمدة سنة
- فريق العمل مقفل الرموز لمدة سنتين
